Pętle

Pętle odpowiadają wykonywanie bloku instrukcji, aż do momentu, gdy dany warunek przestanie być spełniony.
W bashu rozróżniamy 4 rodzaje pętli:

  • while
  • for
  • until
  • select

Pętla while

Składnia:

while [ warunek ] ; do
  polecenia
done

Przykład użycia:

!/bin/bash
x=0
while [ x < 3] ; do
  echo "wartosc x to: $x"
  x=$[x+1]
done
echo "koniec petli"

A oto wynik wykonania skryptu:

$ ./skrypt5.sh
wartosc x to 0
wartosc x to 1
wartosc x to 2
wartosc x to 3
koniec petli
