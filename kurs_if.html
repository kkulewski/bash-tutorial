Zmienne warunkowe

Tak jak w innych językach, również w bashu występują typowe instrukcje warunkowe, tj. if-else oraz switch-case. Omówimy tę pierwszą część zagadnienia.

If/Else

Konstrukcja ta sprawdza, czy warunek jest prawdziwy. Jeśli jest, to wykonuje określone instrukcje. Fraza else odpowiada za to, co zostanie wykonane w przeciwnym wypadku i jest opcjonalna.
Przykład:

if (warunek) ; then
  polecenia
else
  polecenia
fi

Przykład

#!/bin/bash
if [ 3 ge 2] ; then
  echo "3 jest większe od 2"
else
  echo"3 nie jest większe od 2"
fi

A oto wynik wykonania powyższego skryptu:

$ ./skrypt4.sh
3 jest większe od 2

Dostępne operatory porównania:

  • eq - równy (equal)
  • ne - różny (not equal)
  • lt - mniejszy (less than)
  • le - mniejszy lub równy (less than or equal)
  • gt - większy (greater)
  • ge - większy lub równy (greater than or equal)
